#Profily

This is a little snippet to replace empty profile picture.

##Usage
###Simple usage
    $('img').profily();

###With option
	$('img').profily({
		fontFamily    : 'Calibri',
		attribute     : 'alt'
	});

You can set an other font family (Calibri by default), and choose an other attribut to get the fullname (alt by default)
